# 💅 OpenGloweauty - Guide Complet

## 🎯 Pour les Professionnels de la Beauté

### 📋 Comment ajouter des prestations

#### Étape 1 : Accéder à votre profil
1. Connectez-vous à votre compte professionnel
2. Cliquez sur **"Profile"** dans le menu de navigation
3. Sélectionnez l'onglet **"Prestations"**

#### Étape 2 : Ajouter une nouvelle prestation
1. Cliquez sur le bouton **"➕ Ajouter une prestation"**
2. Remplissez le formulaire :
   - **Nom** : Ex: "Coupe femme", "Balayage", "Manucure complète"
   - **Catégorie** : Choisissez parmi 12 catégories (Coiffure, Manucure, etc.)
   - **Description** : Décrivez votre prestation en détail
   - **Durée** : En minutes (ex: 60)
   - **Prix** : En euros (ex: 35.00)
3. Cliquez sur **"➕ Ajouter"**

#### Étape 3 : Gérer vos prestations
- **Modifier** : Cliquez sur l'icône ✏️ pour éditer une prestation
- **Activer/Désactiver** : Cliquez sur l'icône 👁️ pour rendre visible ou cacher une prestation
- **Supprimer** : Cliquez sur l'icône 🗑️ pour supprimer définitivement

## 👤 Pour les Clients

### 🔍 Comment réserver

#### Étape 1 : Rechercher un professionnel
1. Allez sur la page **"Recherche"**
2. Utilisez la barre de recherche pour trouver :
   - Un nom de salon
   - Une adresse
   - Une spécialité
3. Filtrez par profession (Coiffeur, Esthéticien, etc.)

#### Étape 2 : Consulter le profil
1. Cliquez sur le professionnel qui vous intéresse
2. Découvrez :
   - Les prestations disponibles
   - Les tarifs et durées
   - La galerie photos
   - Les informations de contact
   - Les horaires d'ouverture

#### Étape 3 : Réserver
1. Dans la section **"📅 Réserver"** :
   - Choisissez une prestation
   - Sélectionnez une date
   - Choisissez une heure
2. Vérifiez le récapitulatif (durée, prix)
3. Cliquez sur **"Confirmer la réservation"**

#### Étape 4 : Gérer vos réservations
1. Allez sur **"📅 Mes Réservations"**
2. Consultez toutes vos réservations :
   - En attente
   - Confirmées
   - Terminées
3. Annulez si besoin avant la date
